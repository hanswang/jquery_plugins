Fading with Autoplay

Stop the autoplay by hovering the slideshow 

          
$("#example-1").okCycle({
  autoplay: true,
  effect:'fade', 
  ui: ['caption','navigation','pagination','currentPage']
});

Scrolling with Captions

Try using your mousewheel

            
$("#example-2").okCycle({
  effect:'scroll',
  ui: ['caption','navigation','pagination','currentPage', 'mouseWheel']
});

The normal behavior is to use the elements data-caption attribute to provide captions. However, if the data-caption is prefixed with an octothorpe (#), it will be treated as a selector and the innerHTML for the corresponding element will be used.

Sliding with Deferred Loading

Image loading can be deferred until a given item becomes visible.

Additionally we are making use of image loading callbacks in order to show loading progress 

          
$("#example-3").okCycle({
  effect:'slide', 
  loadOnShow: true, 
  ui: ['caption','navigation','pagination','currentPage'],
  onProgress: function(data, img){
    var width = (data.loaded / data.total) * 100;

    $(img).fadeIn();

    $("#progress > div").css({width: width + '%'});
  },
  onDone: function(options){
    setTimeout(function(){
      $("#progress > div").css({backgroundColor: '#fff'}).html("All loaded");
    },500);
  }
});
